# 执行代码
```
Intent intent = new Intent();
intent.setAction(Settings.ACTION_SETTINGS);
try {
    activity.startActivity(intent); 
} catch (ActivityNotFoundException e) {
    // 可能对应的界面不存在，抛出ActivityNotFoundException，导致崩溃
    e.printStackTrace();
}
```
